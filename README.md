# DT - Language Comparison Tool

## About This Project

As a librarian with a passion for languages, I've always been fascinated by how different languages express the same ideas. This tool allows you to compare sentences across Danish, Swedish, English, German, and French - languages from countries I've had the pleasure of traveling to and experiencing firsthand.

## My Background

I work as a librarian and have a curious mind that loves exploring languages and cultures. Through my travels to Denmark, Sweden, Germany, France, and English-speaking countries, I've developed an appreciation for the similarities and differences between these languages. This website is a fun way to compare and learn!

## Features

- Compare sentences across 5 languages
- Simple, clean interface
- Pre-loaded common phrases
- Easy to use for language learners and enthusiasts

## How to Use

Simply enter a Danish sentence and see how it translates across the other languages. Perfect for language learning, comparison, and satisfying linguistic curiosity!
